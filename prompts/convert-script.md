# Script Conversion Prompt

Convert the adaptation draft into the requested script format.

Supported formats:

- `comic_narration`: panel, narration, dialogue, sound, visual action.
- `hollywood`: scene heading, action, character, dialogue, parenthetical.

Rules:

- Do not invent major plot events unless requested.
- Keep character names consistent.
- Make visual action concrete enough for storyboard generation.
- Flag scenes that require manual continuity review.
